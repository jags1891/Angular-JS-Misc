Slide Animation

instead of passing a number to animate() funtion, we can pass a string.
Ex: '0.5s 1 ease-in' or '0.5s 1 cubic-bezier(.17, .67, .83, .67)'
0.5s - animation time
1 - 1sec delay
ease-in- Easing (a function that determines speed of animation over time)
In CSS, we have 3 types of easings -
1) Standard easing - Linear speed (same speed).
2) Ease-in - Start slowly, end fast. (move out)
3) Ease-Out- Starts fast and ends slow. (move in)
4) Ease-in-out - Starts slowly, picks up speed and ends slowly.
Cubic Be`zier : cubic-bezier(.17, .67, .83, .67)
