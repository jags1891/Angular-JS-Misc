steps to setup Firebase

  1. install Firebase: run -> npm install firebase angularfire2 -- save
  2. Add a property "Firebase" in environment.ts add settings copied from the firebase website
  3. Add the following to app.module.ts
    • {{step1}}
    • imports: [ BrowserModule, AngularFireModule.initializeApp(environment.firebase), AngularFireDatabaseModule ]
  4. Inject AngularFireDatabase into the componet you want to use.
